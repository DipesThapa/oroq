  1. Tap Allow when iOS asks "Allow profile download?"

  3. Tap Profile Downloaded near the top (or Settings → General → VPN & Device Management)
  4. Tap Install in the top right
  5. Enter your iPhone passcode
  6. Tap Install twice more to confirm

Android 9 (Pie) or newer supports DNS-over-TLS natively. No app needed — just one setting change.

Copy this hostname:

kids.dns.cyberheroez.co.uk
  1. Open Settings on the device
  2. Tap Network & Internet → Private DNS (on some phones: Connections → More → Private DNS)
  3. Select Private DNS provider hostname
  4. Paste the hostname above
  5. Tap Save
